Зробити наступний компоненти:
  Header - компонент хедеру, має містити заголовок сайту і навігаційний список
  Footer - компонент футеру, має містити текст про про права
  UserProfile - компонент профіля користувача. Відображає дані користувача
    Структура об'єкту користувача:
      id: string
      firstName: string
      lastName: string
      imgSrc: string(img name / URL)
      isMale: boolean
      email: string
      password: string

Всі компоненти мають бути стилізовані через всої файли стилів.
